LuaJIT is a Just-In-Time Compiler for Lua 5.1.

LuaJIT is based on » Lua, a powerful, light-weight programming language. LuaJIT extends Lua with a JIT compiler, a set of new API Functions, several add-on modules and accompanying documentation.

LuaJIT depends on Coco to allow yielding from coroutines for JIT compiled functions. Coco is part of the LuaJIT distribution.
